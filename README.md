## スキル・技術

### 言語・フレームワーク
`Python` / `JavaScript` / `TypeScript` / `Swift` / `Flutter` / `Node.js` / `Next.js`

### DevOps・サーバー管理
- Linux（UbuntuServer、systemd）、Proxmox VE、Docker、k3s、Windows Server

- DB：PostgreSQL、MySQL、Redis

### 自動化・監視
